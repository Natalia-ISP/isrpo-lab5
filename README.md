## Теги в HTML

cтруктура парного тега:
**<тег>тело</тег>**
Пример:
```html
<h1>заголовок</h1>
```

## Базовые HTML-теги
### Примеры:
```html
<h2>Заголовок</h2>
<p>Абзац текста</p>
<strong>Жирный</strong>
<em>Курсив</em>
<hr>
<a href="https://example.com">Ссылка</a>
<img src="example.jpg" alt="Описание">
```
